Step 1 — Get your API key
1. Go to resend.com → Sign up free (use Google)
2. Click API Keys in left sidebar
3. Click Create API Key → name it InvoicePro → copy it

Step 2 — Add your domain (send from invoicepro@asproite.com)
1. Go to resend.com/domains
2. Click Add Domain → type asproite.com
3. Copy the 3 DNS records shown
4. Go to Hostinger hPanel → Domains → DNS Zone
5. Add the 3 records → click Verify in Resend
💡 DNS verification takes 1–5 minutes usually. Once verified, you can send to any email.
